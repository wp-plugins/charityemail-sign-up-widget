Add a "Subscribe to Newsletter" widget to your WordPress site that will insert your contact in one of your CharityeMail Address Books.

== Description ==

Add this widget to your site to allow your visitors to sign up to your e-newsletter and email marketing campaigns, sent using the CharityeMail email marketing system. The email address of your new subscriber will be added to the CharityeMail Address Book you specify within your Settings in Wordpress. If you're not already a CharityeMail user you can find out more about CharityeMail at http://www.charityemail.co.uk.

== Installation ==

1. Search for the plugin 'CharityeMail Sign Up Widget'.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click Settings in your WordPress admin section.
4. Select 'CharityeMail Sign Up Widget'.
5. You will be prompted to enter your API key details, these can be obtained in your CharityeMail account by doing the following:
	1. Navigate to the 'My Account' section in CharityeMail.
	2. Click 'Manage Users'.
	3. Select the API tab and click Add Account
	4. Copy the generated email address without editing it.
	5. Enter in a password.
	6. This will be your API Email and API Password.

6. Click 'Connect and list Address Books'.
7. This will display a list of address books in your CharityeMail account.
8. Select the address book you wish contacts to be uploaded into (Only one may currently be selected).
9. If you wish, you can include datafields in the form (Firstname - Lastname - Fullname are available).Datafields will NOT be required input fields.
10. Click save settings.  

The widget can now be added to your website. To do this follow the steps below:

1. Select Appearance in your WordPress admin section
2. Select Widgets
3. Locate the CharityeMail Sign Up Widget in your available widgets
4. Drag the widget where you want it to appear on your WordPress powered website.

== Frequently Asked Questions ==

Q. Can I select more than one address book to sign contacts up to?

A. Currently you may only select one Address Book to place contacts into.

Q. My contacts are not appearing in my Address Book?

A. Check you have followed the steps exactly as above and that your API email and API password appear exactly as they do in your CharityeMail account.  Remember that the API email is automatically generated and should not be changed.
